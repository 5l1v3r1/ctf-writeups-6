# Intro to Stegano 2

For this challenge we are provided with an image `chall.jpg`.

The flag is encoded in the windows of the huge building on the far left in binary ASCII. A bright window represents a one, a dark window a zero:

```
01000011 C
01010011 S
01000011 C
01000111 G
01111011 {
01100001 a
01011111 _
01000110 F
01101100 l
00110100 4
01100111 g
01111101 }
```

Flag: `CSCG{a_Fl4g}`
